# Vue-express-demo

## install dependencies
``` bash
cd client
npm i
```

``` bash
cd server
npm i
```

## run mongodb
``` bash
# win
C:\Program Files\MongoDB\Server\3.4\bin\mongod.exe

# mac
mongod
```

## development
``` bash
# serve at localhost:1024
cd client
npm run dev
```

``` bash
# serve at localhost:2048
cd server
npm run start
```
